Eventos en Javascript (Edicion botones y enlaces)

Cada elemento de HTML tiene definida su propia lista de posibles eventos que se le pueden asignar. Un mismo tipo de evento (por ejemplo, pinchar el botón izquierdo del ratón) puede estar definido para varios elementos HTML y un mismo elemento puede tener asociados diferentes eventos.

La diferencia entre un boton y un enlace es que el boton no desencadena ninguna accion por defecto, mientras que un enlace sin importar el evento que se le de, al pulsar con click izquierdo, se te lleva a la pagina que tenga el enlace en este caso, el boton a la derecha de este parrafo, te llevaria a youtube.com

Y, sin embargo, al pulsar con el click izquierdo, te quedas en esta pagina, y esto se repite para los otros enlaces. La forma en la que se consigue esto es en el mismo enlace escribir lo siguiente: onclick="return false;" De esta forma, le decimos al enlace que no nos lleve a la pagina de dicho enlace y solo oculta y muestra el texto.
